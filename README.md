# 刷题助手

一个面向多种考试的备考刷题微信小程序，支持章节练习、随机练习、模拟考试和错题本功能，配套 Web 管理后台供管理员录入和管理题库。

## 系统架构

```
┌─────────────┐     ┌──────────────┐
│  微信小程序  │    │  管理后台 Web │
│  (用户刷题)  │    │  (题库管理)   │
└──────┬──────┘     └──────┬───────┘
       │                    │
       └────────┬───────────┘
                │ HTTPS
       ┌────────▼────────┐
       │  Node.js/Express│
       │   REST API      │
       ├─────────────────┤
       │  Sequelize ORM  │
       └────────┬────────┘
                │
       ┌────────▼────────┐
       │     MySQL       │
       └─────────────────┘
```

项目包含三个部分：

| 模块 | 目录 | 技术栈 |
|------|------|--------|
| 微信小程序 | `miniprogram/` | 微信原生（WXML + WXSS + JS） |
| 管理后台 | `webadmin/` | Vue 3 + Element Plus |
| API 服务 | `server/` | Node.js + Express + Sequelize + MySQL |

## 功能特性

### 小程序端

- **章节练习** — 按章节顺序刷题，实时判对错并显示解析
- **随机练习** — 从题库中随机抽题，优先未答过的题目
- **模拟考试** — 按预设配置组卷，限时作答，交卷后统一批改
- **错题本** — 自动收集错题，支持错题重练和标记已掌握
- **学习统计** — 总答题数、正确率、连续学习天数、各章节进度

### 管理后台

- **仪表盘** — 总用户数、今日活跃、总题目数、答题趋势
- **考试分类管理** — 创建和管理不同考试科目
- **章节管理** — 为每个考试科目配置章节
- **题目管理** — 支持单选、多选、判断、填空四种题型，支持批量操作
- **模考配置** — 设置题目数量、时间限制、抽题规则（随机/按章节比例）
- **用户管理** — 查看用户列表及答题统计

## 快速开始

### 环境要求

- Node.js >= 18
- MySQL >= 8.0
- 微信开发者工具

### 后端服务

```bash
cd server

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 填写数据库连接信息和微信 AppID/Secret

# 初始化管理员账号
npm run seed

# 启动开发服务
npm run dev
```

环境变量说明：

| 变量 | 说明 |
|------|------|
| `DB_HOST` | 数据库地址 |
| `DB_PORT` | 数据库端口，默认 3306 |
| `DB_NAME` | 数据库名，默认 quiz_app |
| `DB_USER` | 数据库用户名 |
| `DB_PASSWORD` | 数据库密码 |
| `JWT_SECRET` | JWT 签名密钥 |
| `WECHAT_APPID` | 微信小程序 AppID |
| `WECHAT_SECRET` | 微信小程序 AppSecret |
| `PORT` | 服务端口，默认 3000 |

### 小程序

1. 使用微信开发者工具打开 `miniprogram/` 目录
2. 修改 `utils/api.js` 中的 API 地址指向本地后端服务
3. 编译运行

### 测试

```bash
cd server
npm test
```

## API 概览

### 小程序端 (`/api/v1`)

- **用户** — 微信登录、绑定手机号、获取用户信息
- **题库** — 考试分类列表、章节列表、题目列表
- **练习** — 开始练习、提交答案、结束练习
- **模考** — 开始模考、交卷
- **错题** — 错题列表、标记已掌握、错题重练
- **统计** — 答题概览、分类进度

### 管理后台 (`/admin/v1`)

- **认证** — 管理员登录、修改密码
- **分类/章节/题目** — 增删改查、归档
- **模考配置** — 增删改查
- **用户管理** — 列表、详情
- **数据统计** — 仪表盘数据
